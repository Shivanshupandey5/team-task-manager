# team-task-manager


Task Manager with role-based access (Admin/Member)

🚀 Features
Authentication (JWT)
Role-based access (Admin / Member)
Task creation & assignment
Status tracking
Dashboard (stats + charts)
🛠 Tech Stack
Frontend: React
Backend: Node.js, Express
Database: MongoDB
⚙️ Setup Instructions
# backend
cd backend
npm install
npm run dev

# frontend
cd frontend
npm install
npm start
🔐 Environment Variables
MONGO_URI=
JWT_SECRET=
ADMIN_INVITE_TOKEN=
